CSS Variables

Отображение в браузере:

Переменные в CSS должны быть объявлены в селекторе CSS, который определяет его область действия.
Для глобального охвата используется селектор root или body. Имя переменной должно начинаться с двух тире (-) и чувствительно к регистру.
