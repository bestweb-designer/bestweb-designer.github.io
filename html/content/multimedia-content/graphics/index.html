Элемент svg является контейнером для масштабируемой векторной графики.

Сравнение canvas и svg

Canvas SVG
Зависит от разрешения Не зависит от разрешения
Не поддерживает обработчики событий Поддерживает обработчики событий
Плохие возможности рендеринга текста Лучше всего подходит для приложений с большими областями рендеринга (Google Maps)
Можно сохранить полученное изображение как .png или .jpg При сложном рисунке медленный рендеринг
Хорошо подходит для игр с интенсивной графикой Не подходит для игровых приложений
