Основная структура html файла

Опишем структуру построчно:

1. Декларирует, что это документ HTML5

2. Корневой элемент HTML. Здесь задается язык страницы.

3. Содержит мета-информацию о документе.

4. Стандарт кодировки символов. UTF-8 позволяет корректно отображать английский, русский и другие языки.

5. Viewport - это видимая область страницы. Сообщается для корректного отображения на мобильных устройствах.

6. Определяет заголовок на панели браузера. Используется как заголовок для поисковых систем.

8. Определяет тело документа. Содержит контент HTML документа.
